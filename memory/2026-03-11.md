## 技術筆記

### Chrome Remote Desktop 限制
- 官方不支援真正的「實體桌面同步」模式
- 一定會建立獨立的虛擬桌面（與本機桌面分離）
- 解析度與 DPI 設定會影響本機顯示

### Wayland vs Xorg
- Wayland：現代、安全、但遠端工具支援較差
- Xorg：傳統、遠端友善、但安全性較低
- CRD 在 Xorg 下運作較穩定，但會改變本機解析度

### 技能命名最佳實踐
1. 使用 snake_case（如 `search_github_repo`）
2. 避免空格與句號（不要 `search.github`）
3. 採用「動詞 + 名詞」結構（如 `get_github_issues`）
4. Description 要極度具體，寫給 AI 看的提示詞
